Design Website Layanan Cloud PT Crayon Indonesia

Ini merupakan project terakhir yang saya kerjakan untuk memenuhi tugas besar mata kuliah Perancangan Interaksi. Pada project ini, saya dan kelompok saya mendesign website layanan cloud untuk PT Crayon Indonesia. Fitur e-invoice adalah salah satu fitur yang saya kerjakan, berisikan faktur pembayaran layanan cloud pelanggan. Pada fitur ini, pelanggan dapat memilih berdasarkan preferensi masing masing untuk pengiriman fakturnya, bisa melalui email, sms, atau dengan mengunduh PDF nya.

Design dibuat dengan semenarik mungkin namun kami juga tetap memperhatikan kemudahannya. Kami memastikan agar flow dan tata letak pada website yang kami design mudah dipahami dan tidak membuat pengguna kesulitan.

Pengelola Keuangan SaveTie

SaveTie merupakan sebuah aplikasi mobile yang digunakan untuk mengelola keuangan. Didalamnya, para pengguna dapat mencatat pengeluaran dan pemasukan setiap harinya. SaveTie juga dilengkapi dengan grafik yang menunjukkan persentasi tiap kategori pengeluaran dan pemasukan.
